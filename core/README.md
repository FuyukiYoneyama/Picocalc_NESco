# core/

This directory is not part of the current PicoCalc_NESco build.

It contains an earlier emulator-core line that was intended to become an
MIT-licensed clean-room implementation. That line of development has been
discontinued. The current active emulator core is `infones`.

The discontinued line started from an attempt to write functional
specifications from `infones` behavior and then implement only those
functions. Later review found that the specifications still contained
`infones`-specific names and references, so the project can no longer treat
this directory as a confirmed clean-room emulator core.

This directory is kept only as historical project material. Do not treat it
as an active source tree or as reusable clean-room emulator code without a
separate review.

See `../docs/project/Picocalc_NESco_HISTORY.md` for the project history.
